# Installation localhost


## Install Git and repository

Install development software
```
sudo apt install git curl codium pgmodeler
```

Configure git in development environment
set variable values in []
```
cd $HOME
git clone [repository .git url]
git config user.name "[name]"
git config user.email "[email]"
git remote set-url origin https://[TOKEN]@[git host]/[USERNAME]/[REPOSITORY].git
```    

## Install Node.Js

```
cd $HOME
mkdir node
wget https://nodejs.org/dist/v20.20.0/node-v20.20.0-linux-x64.tar.xz
tar -xJf node-v20.20.0-linux-x64.tar.xz -C node --strip-components=1
echo 'export PATH="$PATH:$HOME/node/bin"' >> ~/.bashrc	
source ~/.bashrc
git clone [repository .git url] app_portfolio
cd app_portfolio
npm install
```
