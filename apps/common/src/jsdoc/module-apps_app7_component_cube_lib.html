(inner) AffineMatrix(e0, e1, e2, e3, e4, e5, e6, e7, e8, e9, e10, e11)

This represents an affine 4x4 matrix, stored as a 3x4 matrix with the last row implied as [0, 0, 0, 1]. This is to avoid generally unneeded work, skipping part of the homogeneous coordinates calculations and the homogeneous divide. Unlike points, we use a constructor function instead of object literals to ensure map sharing. The matrix looks like: e0 e1 e2 e3 e4 e5 e6 e7 e8 e9 e10 e11 0 0 0 1
