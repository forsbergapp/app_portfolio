(inner) getDirectionFromFaces(origin, target, orientation) → {string|undefined}

Finds the direction from an origin face to a target face. The origin face will be oriented so that it becomes FRONT. An orientation object must be provided that specifies any of these faces (exclusively): TOP, RIGHT, DOWN, LEFT. If FRONT or BACK is provided along with one of those faces, it will be ignored. If FRONT or BACK is the only face provided, the orientation is ambiguous and an error will be thrown.

Parameters:
Name Type Description
origin string

The origin face.
target string

The target face.
orientation object

The object that specifies the cube orientation.

Example
getDirectionFromFaces('back', 'up', { down: 'right' })
				Step 1) orient the BACK face so that it becomes FRONT.
				Step 2) orient the DOWN face so that it becomes RIGHT.
				Step 3) Find the direction from BACK (now FRONT) to UP (now LEFT).
				Step 4) Returns 'left'.
