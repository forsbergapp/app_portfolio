Timetables
World's most advanced traditional prayer timetable calculation
This is a progressive web app (PWA) and a single page application (SPA). It works on a mobile or in any supported browser and can be installed in the mobile. All timetable calculations are in real time.
Features
Use provider sign in to login or create a local user to save your settings and then you will also be able to share your timetables on your social network.
Local users use email verification to create account.
all known regional settings, all languages, all countries supported
support for second language
arabic support of different arabic scripts
support of different Hijri calendar types
support of different number systems
right to left layout
timetables in Gregorian or Hijri calendar
show column titles with transliterad text or translated text in any order
add unique design with images, text and different themes to share with others
different ways to highlight rows, Friday, Saturday, Sunday or groups of 10 days
add temporary notes to month report before printing
print or share selected report in HTML or PDF using PDF server generation with support for all regional settings
automatic GPS lookup where you are or selected on map
automatic timezone lookup where you are or selected on map
map with Qibbla
all known methods configurable to calculate times
different asr method
high latitude adjustment
support of Hijri adjustments
fasting information for Ramadan with flexible start and end
optional imsak, sunset and midnight prayer times
iqamat configurable times
multiple places support shown on the day report where each place can show different regional settings
QR code for profiles or user settings that can be shared
like and follow others
view count information on profies and on user settings
search other users
top users
information of follow and like statistics for current logged in user
view other users timetables and like them to keep track of liked timetables
